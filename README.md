# Student Management CRUD Application

A simple CRUD web application built using PHP and MySQL to manage student records.

This project allows users to add, view, update, and delete student information through a simple interface.

## Features

- Add new student records
- View all students
- Edit student information
- Delete student records
- MySQL database integration
- Form validation
- PDO prepared statements for secure queries
- Responsive UI

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP
- phpMyAdmin

## How to Run

1. Install XAMPP
2. Start Apache and MySQL
3. Move project folder to:

```text
C:\xampp\htdocs
```

4. Open phpMyAdmin:

```text
http://localhost/phpmyadmin
```

5. Create a database named:

```text
student_crud
```

6. Import `database.sql`

7. Run project in browser:

```text
http://localhost/student-management-crud
```
